Can be found inside `src/components/FixedPlugin/FixedPlugin.js`. This component
is the right menu and its purpose is so you can easily customize the left menu
and also change the Chakra color mode. It is only rendered inside the
`src/layouts/Admin.js` and `src/layouts/RTL.js` layouts.
