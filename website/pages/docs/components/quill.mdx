---
title: Chakra UI Quill - Horizon UI
description: Renders a Quill Editor with different options.
---

The Text Editor provided by Quill comes with different settings that the user
can easily modify. More info on Quill and how it works
[here](https://www.npmjs.com/package/react-quill) .

## Usage

```jsx
function Example() {
  return (
    <Box bg="red.400" color="white">
      <iframe
        src="https://codesandbox.io/embed/mystifying-parm-4bwdc?fontsize=14&hidenavigation=1&theme=dark&hidedevtools=1&editorsize=0&initialpath=quill-editor&module=%2Fsrc%2Fcomponents%2Fquill.js"
        width="100%"
        height="500px"
        ms="-15px"
        overflow="hidden"
      ></iframe>
    </Box>
  )
}
```
