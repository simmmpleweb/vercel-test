Our kanban is built using react-kanban. You can further read about it
[here](https://www.npmjs.com/package/@asseinfo/react-kanban) .

## Usage

```jsx
function Example() {
  return (
    <Box bg="red.400" color="white">
      <iframe
        src="https://codesandbox.io/embed/nice-water-1v4hzb?fontsize=14&hidenavigation=1&initialpath=%2Fkanban&theme=dark"
        width="100%"
        height="700px"
        ms="-15px"
        overflow="hidden"
      ></iframe>
    </Box>
  )
}
```
