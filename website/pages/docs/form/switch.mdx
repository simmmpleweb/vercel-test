The `Switch` component is used as an alternative for the
[Checkbox](/docs-horizon-ui-chakra/docs/form/checkbox) component. You can switch
between enabled or disabled states.

`Switch` must always be accompanied by a label, and follows the same keyboard
workflow as a `Checkbox`.

## Import

```tsx
import { Switch } from "@chakra-ui/react"
```

## Usage

```jsx
<FormControl display="flex" alignItems="center">
  <FormLabel htmlFor="email-alerts" mb="0">
    Enable email alerts?
  </FormLabel>
  <Switch id="email-alerts" colorScheme="brand" />
</FormControl>
```

## Sizes

Pass the `size` prop to change the size of the `Switch`.

```jsx
<Stack align="center" direction="row">
  <Switch size="sm" colorScheme="brand" />
  <Switch size="md" colorScheme="brand" />
  <Switch size="lg" colorScheme="brand" />
</Stack>
```

```tsx
<Stack align="center" direction="row">
  <Switch size="sm" colorScheme="brand" />
  <Switch size="md" colorScheme="brand" />
  <Switch size="lg" colorScheme="brand" />
</Stack>
```

## Switch background color

You can change the checked background color of the `Switch` by passing the
`color` prop.

```jsx
<Stack direction="row">
  <Switch colorScheme="red" />
  <Switch colorScheme="green" size="lg" />
</Stack>
```

```tsx
<Stack direction="row">
  <Switch colorScheme="red" />
  <Switch colorScheme="green" size="lg" />
</Stack>
```
